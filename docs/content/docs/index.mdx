## New tanstack start+shadcn project

```javascript
pnpm dlx shadcn@latest create --preset "https://ui.shadcn.com/init?base=radix&style=vega&baseColor=stone&theme=stone&iconLibrary=lucide&font=inter&menuAccent=subtle&menuColor=default&radius=small&template=start" --template start
```

To customize style:
https://ui.shadcn.com/create

## Initialize srstack

```bash
npx srstack@latest init
```
- Adds vscode/settings.json
- Updates branding
- Adds shadcn: sidebar sonner
